# 灵创健康首页

一个基于 Vite 构建的医疗科技风格首页项目，品牌已替换为“灵创健康”，包含首页焦点 banner 轮播、解决方案区块、产品能力区块和客户案例展示。

## 本地开发

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```

## 在线发布

仓库已配置 GitHub Pages 自动部署工作流。推送到 `main` 分支后，GitHub Actions 会自动构建并发布站点。
